📈 Zerodha Clone (MERN Stack)

A full-stack stock trading platform inspired
by Zerodha (Kite), built using the MERN stack.
This project demonstrates real-world features 
like authentication, dashboard analytics, order 
management, and responsive UI.


🚀 Features

🔐 User Authentication (Login/Register)
📊 Dashboard with portfolio overview
📈 Real-time stock UI (static/mock or API-based)
🛒 Buy/Sell order functionality
📋 Order history tracking
❤️ Watchlist management
📱 Fully responsive design
🔄 State management


🛠️ Tech Stack

Frontend:
React.js
Redux Toolkit
Bootstrap / CSS
Backend:
Node.js
Express.js
Database:
MongoDB (Mongoose)
